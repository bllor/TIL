reduce 매서드를 이용하여 배열의 합계 구하기

reduce매서드는 배열의 각 요소에 대해 누적된 값을 계산하여 단일 결과를 반환합니다
reduce메서드는 두 개의 인자를 받는 콜백함수와 초기값을 인자로 가집니다.
ex)
const a = [1, 2, 3, 4, 5];
let sum = a.reduce((x, y) => x + y);
배열 a의 누적값을 계산하는 코드 입니다
x,y에서 x는 누적값을 y는 현재 배열요소를 의미합니다. 보통(acc,cur)로 많이 사용합니다.
초기값으로 0을 제공하고, 이 초기값은 x의 초기값으로 설정됩니다.
콜백함수 (x,y)=> x+y는 두 인자를 더한 값을 반환하고, 더한 값은 다음 반복에서 x의 값이 됩니다.
a의 길이만큼 반복되며, 반복이 끝난 뒤 배열의 합을 반환합니다.
